**In the APS framework, function is organisational.** A component has a function because it contributes to the self-maintaining organisation of the system — not because it was selected for that role, and not merely because it has causal effects.

This distinguishes the APS account from:
- **Etiological accounts** (selected-effects): function depends on evolutionary history
- **Causal-role accounts** (Cummins): function is any causal contribution to a capacity

The APS view grounds function in the present organisation of the system, making it applicable to systems without evolutionary history (e.g., synthetic life, hypothetical alien biology).
